# TSVFile
可讀取txt和tsv來呈現資料，適合用背英文單字。
<img width="673" height="479" alt="image" src="https://github.com/user-attachments/assets/f1316c1c-d29e-4fb2-a2bc-0c55bc77a21d" />

功能介紹:

1.開啟.tsv/.txt檔案
可使用(alt+關鍵字)，不需用滑鼠點擊
<img width="135" height="96" alt="image" src="https://github.com/user-attachments/assets/45153e66-c8dc-47aa-b4de-a682d9eb5acb" />


2.讀取單字資料(顯示單字列表、顯示音標、顯示音檔路徑、顯示單字解釋)

3.About視窗 描述作者相關介紹

<img width="433" height="291" alt="image" src="https://github.com/user-attachments/assets/0c599b65-322b-4154-960e-df43a0483a2b" />

4.離開確認提示 防止誤觸離開鍵
